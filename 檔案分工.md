# 檔案分工

這份文件說明目前專案中各檔案負責的範圍，方便後續維護、除錯與新增功能。

## 根目錄

### `index.html`

系統主入口。

負責內容：

- HTML 畫面結構
- 左側選單
- 各功能頁面 panel
- 彈窗 modal 結構
- 外部套件引用
- CSS 與 JavaScript 檔案載入順序

注意事項：

- 不放主要業務邏輯
- 不放大量 CSS
- 不放大量 JavaScript
- 新增頁面時，先在這裡新增 panel 結構，再到 `js/ui.js` 補導覽對應

### `庫存管理系統_README.md`

專案說明文件。

負責內容：

- 系統功能總覽
- 操作流程
- 資料儲存方式
- 備份與還原說明
- 版本規劃
- 維護原則

適合放比較完整的產品說明與開發方向。

### `檔案分工.md`

目前這份文件。

負責內容：

- 說明每個檔案的責任
- 協助判斷修改功能時該去哪個檔案

## CSS

### `css/style.css`

系統樣式檔。

負責內容：

- 全站色彩、字體、間距、陰影等設計變數
- sidebar、topbar、panel、card、table、modal 樣式
- button、badge、alert、toast 等共用元件樣式
- 排程月曆、庫位、圖表區塊等畫面樣式

注意事項：

- 優先沿用現有 class
- 不建議在 HTML 裡新增大量 inline style
- 若要新增元件，盡量建立可重用 class

## JavaScript

### `js/data.js`

資料與共用 helper。

負責內容：

- 全域資料狀態：`items`、`bom`、`logs`、`locations`、`schedules`
- localStorage 讀寫
- JSON 備份與還原
- 預設測試資料
- 資料正規化 `normalizeData()`
- HTML escape `esc()`
- 日期 helper `localDateOnly()`、`localTodayString()`
- 出貨前庫存檢查 `buildShipmentCheck()`

適合修改：

- 資料格式
- 舊資料相容
- 備份還原
- 共用安全處理
- 共用日期處理

### `js/analytics.js`

狀態判斷、統計與總覽。

負責內容：

- 庫存狀態判斷 `getStatus()`
- 狀態 badge
- 採購週期計算
- 消耗預測
- 庫位 helper
- 總覽儀表板渲染
- 指標更新 `updateMetrics()`

適合修改：

- 需採購 / 偏低 / 正常的判斷規則
- 採購建議邏輯
- 總覽頁顯示內容
- 預測耗盡天數邏輯

### `js/inventory.js`

庫存總覽與品項管理。

負責內容：

- 庫存表格渲染
- 搜尋與篩選
- 快速調整庫存
- 新增品項
- 編輯品項
- 刪除品項

適合修改：

- 庫存總覽欄位
- 品項表單欄位
- 品項刪除規則
- 快速調整行為

### `js/location.js`

庫位管理。

負責內容：

- 依品項顯示庫位
- 依庫位顯示品項
- 新增 / 更新 / 清空庫位
- 顯示帳面庫存與庫位合計差異

適合修改：

- 庫位編號規則
- 庫位差異提示
- 庫位查詢方式
- 庫位資料顯示格式

### `js/shipment.js`

出貨、進貨與盤點。

負責內容：

- 出貨產品選單
- 出貨包材消耗預覽
- 一般出貨扣庫
- 進貨登記
- 批次盤點

適合修改：

- 出貨防呆
- 出貨扣庫流程
- 進貨欄位
- 盤點差異處理

### `js/bom.js`

BOM 與用量排行。

負責內容：

- BOM 產品 / 包材選單
- 新增 BOM 對照
- 移除 BOM 對照
- 包材使用率排行

適合修改：

- BOM 欄位
- 損耗率邏輯
- 包裝方式分類
- 用量排行統計方式

### `js/report.js`

報表、紀錄、採購與 Excel 匯入匯出。

負責內容：

- 庫存趨勢圖
- 月報表
- 操作紀錄
- 採購清單
- Excel 匯出
- Excel 匯入

適合修改：

- 月報表格式
- Excel 欄位
- 採購清單欄位
- 操作紀錄顯示
- 圖表統計方式

### `js/schedule.js`

出貨排程。

負責內容：

- 排程狀態判斷
- 今日出貨提醒
- 月曆視圖
- 列表視圖
- 新增排程
- 編輯排程
- 確認排程出貨
- 刪除排程

適合修改：

- 逾期 / 即將到期規則
- 月曆顯示方式
- 排程產品列
- 排程出貨扣庫流程

共用 UI 與初始化。

負責內容：

- panel 標題與副標題
- modal 開關
- Esc 關閉 modal
- toast 提示
- `refresh()`
- 系統初始化

適合修改：

- 新增功能頁導覽

- toast 顯示規則
- 全站 refresh 流程

## 外部套件

目前在 `index.html` 中由 CDN 載入：

- SheetJS / XLSX：Excel 匯入匯出
- Chart.js：庫存趨勢圖
- Google Fonts：字體

若未來要支援離線使用，可以把這些套件下載到本機，例如放在 `vendor/` 目錄。

## 修改功能時怎麼找檔案

常見需求對照：

| 想改的功能 | 優先查看 |
| --- | --- |
| 品項欄位、庫存表格 | `js/inventory.js` |
| 出貨扣庫、進貨、盤點 | `js/shipment.js` |
| 排程月曆、排程出貨 | `js/schedule.js` |
| BOM 對照、包材用量 | `js/bom.js` |
| 庫位顯示與差異 | `js/location.js` |
| 月報表、採購清單、Excel | `js/report.js` |
| 總覽指標、採購週期、預測 | `js/analytics.js` |
| 資料格式、localStorage、備份 | `js/data.js` |
| 導覽、modal、toast、初始化 | `js/ui.js` |
| 樣式與畫面外觀 | `css/style.css` |
| HTML 結構與 panel | `index.html` |
